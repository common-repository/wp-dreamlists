== Description ==

A Wordpress plugin that allows visitors to subscribe to Dreamhost announce lists via a widget, as well as view and manage
the lists in the Wordpress admin panel. Additionally the WP Admin can send HTML emails to the lists using the current 
Wordpress template. 

Think of it as a poor man's version of the Constant Contact service! All due to the Dreamhost API.
